# [u/growgrowgrow-counter](https://www.reddit.com/user/growgrowgrow-counter/) Reddit Bot

**THIS BOT IS CURRENTLY IN DEVELOPMENT**
## What is it?
A reddit bot made to count the number of times someone has said phrases related to [r/sssdfg](https://www.reddit.com/r/sssdfg/)'s Apple April event

## Planned Features
- Call the bot by commenting/replying u/growgrowgrow-counter
- The bot will then respond with a list of how many times the person you replied to/the poster has said phrases related to Apple April
- Use u/growgrowgrow-counter [self] for the bot to count how many times you have said the phrases

## Planned Phrases
- "Grow Grow Grow"
- "The rot consumes"
- "Happy Apple April"
